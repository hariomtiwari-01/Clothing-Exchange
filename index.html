Project Overview

ReWear is a community-driven platform promoting sustainable fashion by encouraging clothing reuse through direct swaps and a point-based redemption system.

  • 🌱 Direct clothing swaps or point-based redemption
  • 🔐 User authentication with email/password
  • 🛠 Admin moderation tools
  • 🎯 Target Audience: Eco-conscious users, thrift lovers, sustainable shoppers
